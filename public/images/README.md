# Hero image drop

Save the dusk outdoor-kitchen hero photo here as:

    hero-outdoor-kitchen.jpg

PhotoHero.astro points at /images/hero-outdoor-kitchen.jpg and animates fire
on it. Flame/heater regions are set in PhotoHero.astro (data-flame-rects /
data-glow-rects) and can be tuned per NEEDS_CALIBRATION.md.
